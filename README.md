# 🧮 Python Calculator (CLI)

A simple command-line calculator built using Python.  
This project performs basic arithmetic operations with user input and includes error handling.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
- Modulo operation
- Continuous loop until user exits
- Input validation (handles invalid input)

---

## 🛠️ Technologies Used

- Python 3
